Subcheat

© 2009 Arjan van der Gaag

A simple wrapper around Subversion’s svn that adds a little extra power.

Subcheat adds a few extra subcommands to svn and sprinkles a wee bit of magic fairy dust om some existing ones. Check it out:

  • svn undo: roll back a revision or range of revisions.
  • svn branch: list, create or delete branches.
  • svn tag: list, create or delete tags.
  • svn rebase: merge changes from trunk back into the current branch.
  • svn reintegrate: merge changes from your branch back into trunk.
  • svn pretty-log: print readable, single-line logs

To get started: install the subcheat Ruby gem: gem install subcheat (not available yet, install manually for now), then alias subcheat to svn: alias svn=subcheat. You can now use all the normal svn-commands – and then some!
